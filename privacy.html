Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pribadi Anda ketika Anda menggunakan aplikasi Pendataan Panen Sawit. Kami berkomitmen untuk melindungi privasi Anda dan memastikan bahwa data pribadi Anda dikelola dengan aman.

1. Pengumpulan Data

Data yang Kami Kumpulkan:

  • Data Identitas: Nama, email, nomor telepon, dan informasi kontak lainnya
  • Data Perusahaan: Nama perusahaan, alamat, dan informasi terkait perkebunan
  • Data Operasional: Data panen, hasil produksi, lokasi kebun, dan data operasional lainnya
  • Data Teknis: Informasi perangkat, alamat IP, jenis browser, dan data log sistem

Metode Pengumpulan:

Input Pengguna

Data yang Anda masukkan secara manual

Upload File

Dokumen dan file yang Anda unggah

Otomatis

Data teknis dan log sistem

Cookie & Analytics

Data penggunaan aplikasi

2. Penggunaan Data

Layanan Utama

Menyediakan dan mengelola layanan pendataan panen sawit

Keamanan

Memastikan keamanan sistem dan mencegah penyalahgunaan

Analisis

Menganalisis penggunaan untuk pengembangan fitur

Dukungan

Memberikan dukungan teknis dan layanan pelanggan

3. Penyimpanan Data

Durasi Penyimpanan

Data Operasional Selama akun aktif
Data Backup 5 tahun
Data Log Sistem 2 tahun

Lokasi Penyimpanan

Data Anda disimpan di server yang berlokasi di Indonesia dengan standar keamanan tinggi. Kami menggunakan penyedia layanan cloud yang mematuhi regulasi perlindungan data Indonesia.

4. Perlindungan Data

Enkripsi Data

Semua data dienkripsi dengan standar AES-256

Firewall

Perlindungan lapisan ganda dari serangan eksternal

Backup Rutin

Backup harian dan pemulihan data otomatis

5. Hak Pengguna

Hak Akses

Mengakses data pribadi Anda yang kami simpan

Hak Perbaikan

Memperbaiki data yang tidak akurat atau tidak lengkap

Hak Penghapusan

Meminta penghapusan data pribadi Anda

Hak Portabilitas

Menerima data Anda dalam format yang terstruktur

Cara Menggunakan Hak Anda

Untuk menggunakan hak-hak di atas, silakan hubungi kami melalui halaman kontak. Kami akan merespon permintaan Anda dalam waktu 14 hari kerja.

6. Penggunaan Cookie

Kami menggunakan cookie untuk meningkatkan pengalaman pengguna dan menganalisis penggunaan aplikasi.

Cookie Penting

Diperlukan untuk fungsi dasar aplikasi

Tidak bisa dinonaktifkan

Cookie Analitik

Untuk analisis penggunaan aplikasi

Bisa dinonaktifkan

Cookie Preferensi

Menyimpan preferensi pengguna

Bisa dinonaktifkan

Cara Mengelola Cookie

Anda dapat mengelola preferensi cookie melalui pengaturan browser Anda. Namun, menonaktifkan cookie penting dapat mempengaruhi fungsi aplikasi.
